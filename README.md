# DaveFolio - Modernized Web Portfolio

Welcome to my completely rebuilt, modern web portfolio! 

This repository showcases my expertise in frontend development, data analytics, and modern AI development concepts. Originally a static HTML/Bootstrap site, it has been completely rewritten into a premium, state-of-the-art **Next.js** application.

## 🌟 Key Features

- **Next.js Architecture:** Built from the ground up using the Next.js App Router for optimal performance, server-side rendering, and seamless page transitions.
- **Dark Glassmorphism Theme:** A sleek, premium dark mode aesthetic with blurred glass surfaces, floating UI cards, and vibrant neon accents.
- **Interactive Network Background:** Features a highly performant `tsparticles` background, symbolizing interconnected multi-agent AI systems and networking.
- **Modern Typography:** Clean, beautiful font pairings using Google Fonts natively (`Outfit` for headings and `Inter` for body text).
- **Responsive Design:** Fluidly adapts to mobile, tablet, and desktop screens with custom CSS Grid and Flexbox layouts (No Tailwind bloat—pure CSS Modules!).

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** CSS Modules with custom Glassmorphism utilities
- **Animations:** `@tsparticles/react` & `@tsparticles/slim`
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

To run this project locally, clone the repository and install the dependencies:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

- `src/app/` - Core Next.js routing, global styles, and fonts.
- `src/components/` - Modular React components (Hero, About, Services, Portfolio, Resume, Contact).
- `public/` - Static assets including images and portfolio screenshots.
- `legacy/` - The previous iteration of the portfolio (kept for archival purposes).
